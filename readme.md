## 简介

本项目是基于 [MusicFree](https://github.com/maotoumao/MusicFree) 的本地化修改版本，移除了所有网络功能，仅保留本地音乐播放能力。

**原项目特性**：插件化、定制化、无广告的免费音乐播放器，目前只支持 Android 和 Harmony OS。

> [!IMPORTANT]
> - 本版本移除了网络搜索、在线歌单、排行榜、插件系统等功能
> - 仅支持本地音乐播放、歌词显示等基础功能
> - 本项目仅供学习研究使用，请勿用于商业用途
> - 详细功能说明请参考原项目：[https://github.com/maotoumao/MusicFree](https://github.com/maotoumao/MusicFree)

## 原项目特性

- 插件化：本软件仅仅是一个播放器，本身**不集成**任何平台的任何音源
- 定制化、无广告：提供浅色、深色模式；支持自定义背景
- 隐私：所有数据存储在本地，不收集个人信息
- 歌词关联：支持将两首歌的歌词关联起来
